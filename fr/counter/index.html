Application de Comptage (Counter)

L’application Counter de Pulattech est un outil simple et épuré pour vous aider à compter. Elle est conçue pour être très facile à utiliser et pour économiser la batterie de votre téléphone.

Fonctionnalités Simples

  • Compter avec les Boutons de Volume Comptez sans même regarder votre écran. Utilisez les boutons de volume sur le côté de votre téléphone pour ajouter ou enlever un nombre.

  • Économise la Batterie L’écran reste sombre, sauf pour afficher les chiffres. Cela consomme très peu de batterie pendant que vous comptez.

  • Verrouiller l’Écran Verrouillez l’écran pendant que vous comptez pour éviter les clics par erreur. Un appui long sur l’icône de cadenas le déverrouille facilement.

  • Vibration Douce Le téléphone vibre doucement à chaque fois que vous comptez pour confirmer que le clic a fonctionné. De plus, l’écran ne s’éteint pas pendant que vous comptez.

  • Gérer Plusieurs Listes Suivez plusieurs choses en même temps. Vous pouvez leur donner des noms, des couleurs, les remettre à zéro ou les supprimer quand vous avez fini.

  • Définir une Limite Si vous voulez compter jusqu à un nombre précis, le téléphone vibre pour vous avertir quand vous l’avez atteint.
